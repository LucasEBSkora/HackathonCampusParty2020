# HackathonCampusParty2020
## Repositório para a plataforma desenvolvida para o campus party 2020
Focando na ODS 16, seu objetivo é simplificar e desburocratizar o acesso a informações sobre como adquirir diversas
documentações necessárias à migrantes, traduzindo-as para os idiomas nativos dos migrantes que vêm ao Brasil, e compilando-as em uma página com linguagem mais acessível.
